=== HANDMATIGE SCHEMA EXPORT - VERPLICHT ===

Open Supabase SQL Editor:
https://supabase.com/dashboard/project/qsimnkmkonleyfqsjctj/sql/new

Voer achtereenvolgens deze 3 queries uit en kopieer de VOLLEDIGE output:

--- QUERY 1: Column definitions ---
SELECT
  table_name,
  column_name,
  data_type,
  is_nullable,
  column_default
FROM information_schema.columns
WHERE table_schema = 'public'
ORDER BY table_name, ordinal_position;

--- QUERY 2: RLS status ---
SELECT
  tablename,
  rowsecurity
FROM pg_tables
WHERE schemaname = 'public';

--- QUERY 3: RLS policies ---
SELECT
  schemaname,
  tablename,
  policyname,
  permissive,
  roles,
  cmd,
  qual,
  with_check
FROM pg_policies
WHERE schemaname = 'public';
