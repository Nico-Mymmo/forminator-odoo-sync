Ontwerpnotities

  • Login is een echte <form>: Enter werkt op beide velden, autocomplete intact voor wachtwoordmanagers.
  • Loginknop tijdens submit: disabled + spinner + "Inloggen…" — geen dubbele submits.
  • Bij 401 blijft het e-mailadres staan en wordt het wachtwoord geselecteerd; bij netwerkfout blijft alles staan.
  • Foutmelding heeft een vaste plek boven de knop en verdwijnt zodra een nieuwe poging start.
  • Welkomstkop toont username → full_name → e-mail (eerste niet-lege).
  • Het grid toont alleen grid-modules; admin en asset-manager blijven utility-knoppen in de navbar.
  • Lege staat is instructief en legt uit dat toewijzing automatisch zichtbaar wordt.
