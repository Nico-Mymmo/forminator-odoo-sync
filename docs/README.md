# Documentation Structure

This directory contains all module-specific documentation for the Forminator-Odoo Sync project.

## Directory Structure

### `/sales-insight-explorer/`
Complete documentation for the Sales Insight Explorer module, including:
- **SALES_INSIGHT_COMPLETE.md** - Comprehensive guide and overview
- **SALES_INSIGHT_EXPLORER.md** - Detailed technical documentation
- **iterations/** - All iteration delivery documents (1-9.4)

### `/crm-leads/`
Documentation for CRM Leads extension functionality:
- CRM_LEADS_EXTENSION_COMPLETE.md
- CRM_LEADS_EXTENSION_TEST.md  
- CRM_LEADS_QUICK_START.md

### `/deprecated/`
Older documentation files and temporary artifacts that may be removed in future cleanup:
- Legacy iteration summaries and progress files
- Forensic analysis documents
- Temporary mapping files
- Old test utilities
