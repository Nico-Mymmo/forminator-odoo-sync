Google Chat-kanalen

Mini-apps kunnen berichten sturen naar een gekoppeld kanaal via window.platform.sendChat(). Koppel hier de webhook-URL van een Chat-ruimte (ruimte-titel → Apps en integraties → Webhooks → nieuwe webhook aanmaken). Die URL verlaat de server nooit -- een mini-app kent een kanaal enkel via zijn naam.
